 RAMP

Rapid Analytics & Model Prototyping

using Python

RAMP data challenges

RAMP data challenges are a great way to prototype machine learning workflows. Predictive solutions (code) submitted by participants are trained on a RAMP server and the scores displayed on a learder board.

Collaborate

The solutions submitted by participants can be made publically available during the 'open phase', at the end of a challenge. Participants can reuse and learn from each other's solutions, accelerating development.

Prototype

RAMP workflow allows 'model blending'. A blend of the best models often achieves a better score than the best single submission.

Learn

RAMP data challenges are a great tool to learn data science! Participants can learn from each other, especially during the 'open phase'.

RAMP packages

The RAMP packages aim to help you define, build, manage and optimize machine learning workflows, great for internal prototyping model workflows in your data science team.

RAMP workflow

The RAMP workflow package consists of a set of reusable tools and scripts to define score types (metrics), workflow elements and prediction types.

RAMP board

The RAMP board package consists of a bundle of modules that allow you to easily setup your own RAMP server and deploy new data challenges.
